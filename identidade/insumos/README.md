# Insumos de Marca — AuditCom

Esta pasta foi criada para organizar as imagens da marca e arquivos de apoio do design da AuditCom.

Coloque aqui os seguintes arquivos:
- `logo.png` (Logo padrão, de preferência em formato transparente/vetorial e cor escura)
- `logo-branco.png` (Logo na versão monocromática branca para uso em fundos pretos/escuros)
- `manual-marca.pdf` (Ou outro arquivo contendo as referências originais de design)
- `favicon.ico` / `favicon.png` (Ícone de navegação do app)
- Demais artes de fundo, texturas ou assets adicionais.

O motor da AuditCom lerá este diretório para renderizar propostas, relatórios e interfaces.
